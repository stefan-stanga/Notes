## 1. Username & Handle Tracking
*Find where a specific handle is used across the web.*
* **Sherlock:** `python3 sherlock username` (Search 300+ sites)
* **WhatsMyName:** [whatsmyname.app](https://whatsmyname.app/) (Web-based alternative)
* **Namechk:** Great for checking major social platforms and domain availability.

## 2. Email Discovery & Verification
* **Hunter.io:** Find professional email patterns for a company.
* **Verify-Email.org:** Check if an email address is valid/active.

## 3. Breach Data (Checking for Leaks)
* **Have I Been Pwned:** Check if an email has been compromised in a breach.

## 4. Social Media Investigation
* **Instagram:** Use `instaloader` to scrape profile metadata.
* **Twitter/X:** Use `Twint` (if working) or Advanced Search `(from:username) "keyword"`.
* **LinkedIn:** Search `site:linkedin.com/in/ "Company Name"` via Google.
